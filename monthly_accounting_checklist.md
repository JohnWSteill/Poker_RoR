# Monthly Poker Accounting Checklist
_End of month or pay-period ritual_

## 1. Session Results
- [ ] Enter all completed sessions into `Results_Sept25_26`
- [ ] Verify each session’s buy-in, cash-out, and net
- [ ] Confirm hours logged are accurate
- [ ] Sanity-check monthly totals (net, hours, hourly)

## 2. Expenses
- [ ] Append all poker-related expenses to `2025_Expenses`
- [ ] Tag each expense with:
  - Category (gas, lodging, food, rake/fees, supplies, etc.)
  - Payment method (household, allowance, poker acct, BTC, site)
- [ ] Mark reimbursable expenses
- [ ] Execute reimbursements if applicable

## 3. Bankroll Reconciliation
- [ ] Add a new end-of-period snapshot row to `Allocation`
- [ ] Update balances for:
  - Cash
  - Chips
  - Bank accounts
  - BTC (with valuation date)
- [ ] Reconcile:
  - Prior snapshot
  + Net session results
  − Expenses
  ± Transfers
- [ ] Investigate and resolve any discrepancies

## 4. Allocation & Paycheck
- [ ] Compare current bankroll to target allocation policy
- [ ] Execute transfers if rebalancing is needed
- [ ] Transfer poker “paycheck” if applicable

## 5. Reflection & Close
- [ ] Write a 1–2 sentence reflection:
  - What went well?
  - What needs attention next month?
- [ ] Mark the month as closed
- [ ] Set reminder for next accounting run
